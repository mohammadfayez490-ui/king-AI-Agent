# Project Structure

Purpose:

Define how AstraMind organizes projects.

Structure:

Each project contains:

- Project information
- Files
- Tasks
- Settings
- History


Project Management:

Features:

- Create new projects
- Edit projects
- Archive projects
- Track progress
- Organize resources


Project Status:

- New
- Active
- Completed
- Archived


Goal:

Create an organized project workspace for AstraMind users.
