# Model Router

Purpose:

Select the best AI model for each task.

Routing Rules:

Text Tasks:

- Use language models
- Generate answers
- Analyze information


Image Tasks:

- Use image models
- Create and edit visuals


Audio Tasks:

- Use audio models
- Process voice and sound


Selection:

- Choose based on user request
- Balance quality and speed
- Use available resources

Goal:

Create an intelligent model selection system for AstraMind.
