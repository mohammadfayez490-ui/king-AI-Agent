# API Structure

Purpose:

Define AstraMind API system.

Responsibilities:

- Receive requests
- Connect frontend with backend
- Communicate with agent
- Return responses


Main Endpoints:

Chat:

- Send message
- Receive AI response


Users:

- Login
- Profile
- Preferences


Files:

- Upload files
- Manage files


Projects:

- Create projects
- Update projects


System Flow:

Client

↓

API

↓

Backend

↓

Agent

↓

Response


Goal:

Create the communication layer for AstraMind.
