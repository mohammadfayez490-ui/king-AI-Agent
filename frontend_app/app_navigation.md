# App Navigation

Purpose:

Define navigation structure for AstraMind application.

Main Pages:

Home:

- Overview
- Quick actions


Chat:

- AI conversation
- History


Projects:

- Create projects
- Manage projects


Files:

- Upload files
- View documents


Profile:

- Account information
- Preferences


Settings:

- App configuration
- Security options


Navigation Flow:

Open App

↓

Home

↓

Select Feature

↓

Use System

↓

Return Result


Goal:

Create simple and organized navigation for AstraMind.
