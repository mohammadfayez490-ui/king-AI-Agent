# Frontend Components

Purpose:

Define reusable interface components for AstraMind.

Components:

## Chat Component

Features:

- Message display
- User input
- AI responses
- Conversation history


## File Component

Features:

- Upload files
- Preview documents
- Manage files


## User Component

Features:

- User profile
- Account information
- Preferences


## Tool Component

Features:

- Show available tools
- Run AI actions
- Display results


## Navigation Component

Features:

- Move between pages
- Application menu


Goal:

Create a modular and scalable frontend system.
