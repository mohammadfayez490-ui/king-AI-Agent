# Memory Rules

Purpose:

Define safe rules for AstraMind memory.

Rules:

- Save only useful information.
- Do not store private information without permission.
- Allow users to control their memory.
- Keep memory organized.
- Remove outdated information when needed.

Security:

- Protect stored data.
- Respect user privacy.
- Avoid unnecessary storage.

Assistant Behavior:

- Use memory only to improve helpfulness.
- Do not reveal private memory.
- Ask when memory is unclear.

Goal:

Create a trusted and safe AI memory system.
