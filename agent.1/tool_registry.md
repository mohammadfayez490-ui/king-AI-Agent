# Tool Registry

Purpose:

Register all AstraMind available tools.

Available Tools:

- Education Tool
- File Tool
- Student Assistant
- Developer Tool
- Content Tool
- Media Tool
- Audio Tool
- Language Tool
- Search Tool
- Account Tool

Rules:

- Each tool must have a clear purpose.
- Tools can be added or updated.
- Agent selects tools based on user requests.

Goal:

Create a connected AI tool ecosystem.
